# Iosevka Kian

This repository contains pre-built TrueType font files for a custom variant of the [Iosevka](https://github.com/be5invis/Iosevka) programming font. The fonts were generated using a build plan by **Kian Kasad**, hence the font family name **Iosevka Kian**.

The original build plan is defined [here](https://github.com/kdkasad/dotfiles/blob/ae6ec4ad76af66f53c76962bfbb5ba60ce4b2e88/.config/setup/iosevka-build-plan.toml), and licensed under MIT.
